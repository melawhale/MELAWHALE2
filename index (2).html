📣 Étape 2 / 3
Connecte Meta Ads
Tes dépenses pub Meta (Facebook/Instagram) seront récupérées automatiquement.
📍 Va sur developers.facebook.com/tools/explorer
1. Sélectionne ton app → clique "Generate Access Token"
2. Ajoute les permissions : ads_read + ads_management
3. Copie l'Access Token généré
4. Récupère ton Ad Account ID dans Ads Manager (format : act_123456789)
